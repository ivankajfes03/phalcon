//Setting up project in phalcon 3.4.4 on windows 7
//win 7, 64-bit, xampp,php 7.2.12, dll phalcon_x86_vc14_php7.0_3.4.4.

STEPS:

1.https://phalcon.io/en-us/download/windows

2.composer global require phalcon/devtools v3.4.3 
//phalcon/devtools v3.4.4 retruns bug, https://github.com/phalcon/phalcon-devtools/issues/1336

3.Go to cmd c:\xampp\htdocs> phalcon create-project phalcon

4.http://localhost/phalcon
